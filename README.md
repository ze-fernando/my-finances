
# My Finances (React Native)

Aplicativo mobile de **gerenciamento de finanças pessoais**, desenvolvido com **React Native**, **TypeScript** e **SQLite**.

O app permite registrar **recebimentos e gastos**, acompanhar o **saldo restante**, visualizar **maior gasto**, **percentual de despesas** e gráficos comparando **entradas e saídas**.

Todos os dados são armazenados **localmente no dispositivo**, permitindo que o aplicativo funcione **100% offline**, sem necessidade de internet.

## ✨ Funcionalidades

* Registro de **recebimentos**
* Registro de **gastos**
* Cálculo automático de:

  * Total recebido
  * Total gasto
  * Saldo restante
* Identificação do **maior gasto**
* Cálculo do **percentual de despesas**
* **Gráfico de gastos vs recebimentos**
* Opção de **ocultar valores financeiros**
* Armazenamento local com **SQLite**
* Funcionamento **offline**
